# jlkcbankproject2

Laravel web application for **JAYALAKSHMI MUTUALLY AIDED COOPERATIVE THRIFT AND CREDIT SOCIETY LTD** — home page, about page, financial services, CTA, FAQ, and footer.

## Tech

- Laravel (PHP)
- Blade views, CSS

## Run

- Point your web server (e.g. XAMPP) to the `public` folder or run `php artisan serve`.
